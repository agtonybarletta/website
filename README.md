# Personal Website

[agtonybarletta.it](https://agtonybarletta.it), a personal website that contains some basic info (who am I, contacts etc) and a small collection of my projects and hobbies.

## Done with

Created using [hugo](https://gohugo.io/) to have an intentionally simple and ultra light website generated from markdown files.
This website is a customized version of [hugo-coder](https://github.com/luizdepra/hugo-coder) theme.

It includes inline [modelviewer](https://modelviewer.dev/) component that allows 3d model preview on the web.

Hosted on Google cloud buckets and managed through a CICD pipeline using Github Actions.
